# create-dallinking-boardgame

Scaffolding tool to create a new dallinking boardgame project.

## Usage

```bash
npm init dallinking-boardgame my-game
# or
npx create-dallinking-boardgame my-game
```

## What's included

- Preconfigured boardgame project structure
- Example game components
- Build and development scripts
- Testing setup
