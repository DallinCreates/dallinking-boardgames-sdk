# @dallinking/boardgame-devkit

Development tools and utilities for dallinking boardgames.

## Installation

```bash
npm install @dallinking/boardgame-devkit
```

## Usage

```javascript
import { initDevkit } from '@dallinking/boardgame-devkit';

const devkit = initDevkit();
```

## CLI

```bash
boardgame-devkit --help
```
